# Zadarma SIP Trunking Setup

This guide walks you through connecting Zadarma to Konnect-U so your AI assistant can make and receive calls using your existing Zadarma phone numbers.

## Step 1: Create a SIP extension in Zadarma

1. Log in to your Zadarma account
2. Navigate to your SIP settings and create or choose a SIP extension
3. Note your SIP extension number, username, and password

## Step 2: Configure inbound routing in Zadarma

1. Open your Zadarma phone number settings
2. Configure the call forwarding / SIP routing rules
3. Set the destination to Konnect-U’s SIP server

    <Frame>
      <img
        src="/resources/zadarma-sip-external-server.png"
        alt="Zadarma SIP External Server SIP URI configuration"
      />
    </Frame>

    4. Enter the SIP server address using your **SIP extension number**:

    ```
    YOUR_EXTENSION@5wed79aeavr.sip.livekit.cloud
    ```

    For example, if your extension is `247000`:

    ```
    247000@5wed79aeavr.sip.livekit.cloud
    ```

    <Frame>
      <img
        src="/resources/zadarma-sip-uri.png"
        alt="Zadarma SIP URI"
      />
    </Frame>

    5. Click **Show IP** below the server address to reveal Zadarma's IP addresses (e.g., `185.45.152.216`, `185.45.155.33`)

    <Frame>
      <img
        src="/resources/zadarma-sip-show-ip-button.png"
        alt="Zadarma SIP Show IP button"
      />
    </Frame>

    <Frame>
      <img
        src="/resources/zadarma-sip-show-ip-addresses.png"
        alt="Zadarma SIP IP addresses revealed"
      />
    </Frame>

    <Note>
    **Save these IPs** — you will need them in Step 3 when configuring inbound authorization in the platform.
    </Note>

    ## Step 3: Import Numbers to the Platform

    Now import the number so the platform knows how to route calls.

    1. Go directly to the Phone Numbers page in your dashboard:

       **[https://app.konnect-u.com/phone-numbers](https://app.konnect-u.com/phone-numbers)**

       If it takes you to the login page, log in first — it will automatically redirect you to the Phone Numbers page.

    2. Click **"Integrate SIP trunk"** (top-right corner of the page)

    <Frame>
      <img
        src="/resources/zadarma-import-sip.png"
        alt="Import numbers via SIP trunk"
      />
    </Frame>

    3. Fill in your phone number and SIP credentials:

    ```
    Phone number: e.g., +1 555 0123
    Your SIP Extension: Your SIP extension number (e.g., 247000)
    Username: Your SIP username (the same as your extension e.g., 247000)
    Password: Your SIP password
    ```

    <Frame>
      <img
        src="/resources/zadarma-sip-auth-integration.png"
        alt="SIP Direct credentials"
      />
    </Frame>

    4. Enter the **Server address** for outbound calls:

    ```
    sip.zadarma.com
    ```

    <Frame>
      <img
        src="/resources/zadarma-sip-outbound-settings.png"
        alt="Zadarma SIP outbound server address"
      />
    </Frame>

    5. Select the **Authorization type**. Select **IP** and add the Zadarma IP addresses you noted earlier from **Show IP** in Step 2.

    <Frame>
      <img
        src="/resources/zadarma-ip-inbound.png"
        alt="Zadarma IP Authorization"
      />
    </Frame>

    6. Select the **country** where your SIP trunk is located.

    <Frame>
      <img
        src="/resources/zadarma-sip1-country.png"
        alt="SIP Country selection"
      />
    </Frame>
